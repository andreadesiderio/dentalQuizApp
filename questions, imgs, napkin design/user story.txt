USER STORY

* The starting screen should have a button that users can click to start the quiz

* Users should be asked the questions one by one

* Users cannot skip questions

* Users should be able to see which question they are on such as '2 out of 5' and their current score

* After submitting the answer: 
   1. the user should be given feedback. If they are correct or if they are wrong with the correct answer
   2.the user needs to click an element to move on

*After the last feedback: 
   1. the user should be shown the ending score with a message
   2. the user should be able to take the quiz over
